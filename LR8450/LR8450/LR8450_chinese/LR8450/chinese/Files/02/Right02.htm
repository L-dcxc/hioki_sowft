IEEE 488.2 标准命令

 

命令

数据
（查询时为响应数据）

说明

*CLS

-

清除状态字节和相关队列

*ESR?

A<NR1> = 0 to 255

查询并清除SESR

*IDN?

A$,B$,C$,D$

A$ = Manufacturer's name
B$ = Model name
C$ = Serial number
D$ = Software version

查询设备ID。

*OPC

-

处理完成后设置SESR的LSB

*OPC?

A<NR1>

处理完成后返回ASCII 1

*OPT?

A1,A2,A3,..,A11<NR1> = 0 to 10

查询设备选项配置

*RST

-

初始化仪器设置

*STB?

A<NR1> = 0 to 255

读取STB

*TST?

A<NR1> = 0, 1

查询内存检查的结果

*WAI

-

处理完成后执行以下命令

:ESR0?

A<NR1> = 0 to 255

查询ESR0
