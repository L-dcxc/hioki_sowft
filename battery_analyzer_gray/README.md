# 电池分析软件 - 灰色专业版

## 特点
- **灰色商务风格**：深灰背景，橙色强调色
- **上下分栏布局**：上方全宽波形显示，下方控制面板+KPI监控
- **专业感强**：适合工业环境，减少眼睛疲劳
- **紧凑设计**：充分利用屏幕空间，信息密度高

## 布局特色
- **顶部工具栏**：应用标题和系统操作按钮
- **上方波形区**：三元电池和刀片电池波形左右并排显示
- **下方控制区**：左侧控制面板（产品信息、显示设置、开始按钮），右侧KPI监控（2x2网格布局）
- **底部状态栏**：实时状态信息显示

## 运行
```bash
python -m battery_analyzer_gray.main
```

## 功能
- 全宽波形显示，视觉冲击力强
- 控制面板和KPI监控分离，操作便捷
- 橙色主题高亮，专业商务感
- 适合长时间使用和监控场景
