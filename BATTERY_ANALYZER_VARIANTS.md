# 电池分析软件 - 三种UI变体说明

本项目提供了**三个不同风格**的电池电压与温升分析软件，供客户选择。

## 🎨 三种变体对比

| 特性 | 深色科技版 | 白色简洁版 | 灰色专业版 |
|------|-----------|-----------|-----------|
| **主色调** | 深蓝黑 | 纯白 | 深灰 |
| **强调色** | 青色 | 蓝色 | 橙色 |
| **布局** | 左侧控制栏 | 顶部控制栏 | 卡片式网格 |
| **波形排列** | 上下排列 | 左右并排 | 上下分卡片 |
| **适用场景** | 科技展示、演示 | 办公室、文档 | 工业现场、长时间 |
| **视觉风格** | 科技感、炫酷 | 简洁、现代 | 专业、商务 |

---

## 1️⃣ 深色科技版 (battery_analyzer)

### 特点
- 深蓝黑背景 + 青色高光
- 左侧垂直控制面板
- 双波形上下排列
- 自定义颜色选择器（中文）

### 启动方式
```bash
# Windows
run_dark.bat

# 或直接运行
python -m battery_analyzer.main
```

### 适合场景
- 科技展会、产品演示
- 暗环境下使用
- 强调视觉效果

---

## 2️⃣ 白色简洁版 (battery_analyzer_white)

### 特点
- 纯白背景 + 蓝色主题
- 顶部横向控制栏
- 波形左右并排
- 大号绿色"开始采集"按钮

### 启动方式
```bash
# Windows
run_white.bat

# 或直接运行
python -m battery_analyzer_white.main
```

### 适合场景
- 办公室环境
- 需要打印报告
- 明亮环境下使用

---

## 3️⃣ 灰色专业版 (battery_analyzer_gray)

### 特点
- 深灰背景 + 橙色强调
- 卡片式布局（左控制右波形）
- 网格式信息组织
- 专业商务风格

### 启动方式
```bash
# Windows
run_gray.bat

# 或直接运行
python -m battery_analyzer_gray.main
```

### 适合场景
- 工业现场
- 长时间监控
- 减少眼疲劳

---

## 🔧 共同功能

所有三个版本均支持：

✅ 三元电池 vs 刀片电池实时对比  
✅ 双Y轴显示（电压 0-10V，温度 0-260°C）  
✅ 虚拟数据实时更新  
✅ 4个KPI实时显示  
✅ 产品信息输入  
✅ 开始/停止控制  
✅ 波形颜色自定义（深色版）

## 📦 依赖安装

```bash
pip install PySide6 pyqtgraph numpy
```

---

## 🚀 快速体验

1. 克隆项目
2. 安装依赖
3. 双击对应的 `.bat` 文件或运行 Python 命令
4. 点击"开始"按钮查看实时波形

## 💡 选择建议

- **展示/演示** → 选深色科技版
- **日常办公** → 选白色简洁版
- **工业监控** → 选灰色专业版

客户可以分别试用三个版本，选择最符合使用场景和审美偏好的版本！
