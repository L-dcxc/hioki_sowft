# 电池分析软件 - 三种UI变体说明

本项目提供了**三个不同风格**的电池电压与温升分析软件，供客户选择。

## 🎨 三种变体对比

| 特性 | 深色科技版 | 白色版 | 灰色专业版 |
|------|-----------|-------|-----------|
| **主色调** | 深蓝黑 | 纯白 | 深灰 |
| **强调色** | 青色 | 蓝色 | 橙色 |
| **布局** | 左侧控制栏 | 顶部控制栏 | 上下分栏 |
| **波形排列** | 上下排列 | 左右并排 | 全宽上方 |
| **控制面板** | 左侧垂直 | 顶部横向 | 下方左侧 |
| **KPI显示** | 底部横向 | 底部横向 | 下方右侧网格 |
| **特色功能** | 颜色自定义 | 📈 mX+b + ⚙️ 参数设置 | 专业监控 |
| **适用场景** | 科技展示、演示 | 办公室、文档 | 工业监控、生产线 |
| **视觉风格** | 科技感、炫酷 | 简洁、现代 | 专业、商务 |

---

## 1️⃣ 深色科技版 (battery_analyzer)

### 特点
- 深蓝黑背景 + 青色高光
- 左侧垂直控制面板
- 双波形上下排列
- 自定义颜色选择器（中文）

### 启动方式
```bash
# Windows
run_dark.bat

# 或直接运行
python -m battery_analyzer.main
```

### 适合场景
- 科技展会、产品演示
- 暗环境下使用
- 强调视觉效果

---

## 2️⃣ 白色版 (battery_analyzer_white)

### 特点
- **白色现代风格**：纯白背景 + 蓝色主题色
- **顶部控制栏**：一键式操作，功能丰富
- **左右波形并排**：直观对比显示
- **双功能设置**：mX+b + 系统参数配置

### 布局特色
- **顶部控制栏**：产品信息录入、颜色选择、mX+b、参数设置、开始采集按钮
- **中间波形区**：三元电池和刀片电池左右并排，全宽显示
- **底部KPI栏**：四个实时数值水平排列
- **双重设置**：
  - 📈 **mX+b**：线性校准公式配置
  - ⚙️ **参数设置**：设备连接、数据采集、显示选项配置

### 启动方式
```bash
# Windows
run_white.bat

# 或直接运行
python -m battery_analyzer_white.main
```

### 适合场景
- 办公室环境和文档处理
- 需要系统配置和打印报告
- 明亮环境下长时间使用

---

## 3️⃣ 灰色专业版 (battery_analyzer_gray)

### 特点
- **灰色商务风格**：深灰背景，橙色强调色
- **上下分栏布局**：上方全宽波形显示，下方控制面板+KPI监控
- **专业感强**：适合工业环境，减少眼睛疲劳
- **紧凑设计**：充分利用屏幕空间，信息密度高

### 布局特色
- **顶部工具栏**：应用标题和系统操作按钮
- **上方波形区**：三元电池和刀片电池波形左右并排显示，全宽布局视觉冲击力强
- **下方控制区**：左侧控制面板（产品信息、显示设置、开始按钮），右侧KPI监控（2x2网格布局）
- **底部状态栏**：实时状态信息显示

### 启动方式
```bash
# Windows
run_gray.bat

# 或直接运行
python -m battery_analyzer_gray.main
```

### 适合场景
- 工业监控和生产线
- 长时间数据监控
- 需要全屏波形显示的场景

---

## 🔧 共同功能

所有三个版本均支持：

✅ 三元电池 vs 刀片电池实时对比  
✅ 双Y轴显示（电压 0-10V，温度 0-260°C）  
✅ 虚拟数据实时更新  
✅ 4个KPI实时显示  
✅ 产品信息输入  
✅ 开始/停止控制  
✅ 波形颜色自定义（深色版）

## 📦 依赖安装

```bash
pip install PySide6 pyqtgraph numpy
```

## 🚀 快速体验

1. 克隆项目
2. 安装依赖
3. 双击对应的 `.bat` 文件或运行 Python 命令
4. 点击"开始"按钮查看实时波形

## 💡 选择建议

- **展示/演示** → 选深色科技版
- **日常办公** → 选白色版
- **工业监控** → 选灰色专业版

客户可以分别试用三个版本，选择最符合使用场景和审美偏好的版本！
