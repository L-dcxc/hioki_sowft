# 电池分析软件 - 白色版

## 特点
- **白色现代风格**：白色背景，蓝色主题色
- **布局**：顶部控制栏 + 横向双波形 + 底部KPI
- **现代设计**：扁平化UI，悬停效果明显
- **适合打印**：高对比度，易于导出报告

## 运行
```bash
python -m battery_analyzer_white.main
```

## 功能
- 横向并排对比三元 vs 刀片电池
- 顶部一键式控制栏
- 大号绿色"开始采集"按钮
- 简洁的产品信息输入
- ⚙️ 参数设置功能（设备连接、数据采集参数、显示选项）
- 📈 mX+b功能（支持三元电池和刀片电池）
- 设备连接配置（IP地址、端口、波特率）
- 数据采集设置（采样率、缓冲区、数据格式）
- 显示选项（网格、缩放、图例）
